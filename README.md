 Social Media App (C++ / Qt)

 Overview

This is a desktop-based social media application developed using C++ and Qt framework. The application allows users to create profiles, share posts, interact through comments, and engage with content using a like system.


 Features

* User   login system
* Create, edit, and delete posts
* Like and comment on posts
* User profile management
* Interactive GUI using Qt framework


 Technologies Used

* C++
* Qt Framework
* Object-Oriented Programming (OOP)

 Purpose
 
The project was developed to strengthen understanding of:

* GUI application development
* Object-oriented design principles
* Event-driven programming
